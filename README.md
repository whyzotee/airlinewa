# AirlineWa

This project make for oop in practice 2/2567

To run webite:

Install package first

## requirement

python >= 3.13.1
node >= 22.12.0
pnpm >= 10.6.1

```bash
pnpm install
pip install -r requirements.txt
```

## to start

```bash
pnpm dev
```
